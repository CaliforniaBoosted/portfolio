App Inventor Project

Course: CSE (Computer Science Engineering and Applications : Entry Course)
Duration: September - October 2017
Objective: Design a function game or other application in MIT App Inventor
Type of Project: Partner Project

The App Inventor Project was one of the first projects I did in CSE. This project was supposed to give us an introduction to how programming works via block coding. The rules were to pick a tutorial off of the MIT App Inventor website, but my partner and I decided that we didn't want to do that, thus we decided on re-creating Atari Breakout from scratch. This was a really fun project and it taught me a lot about coding. You can see our official project notebook here.
